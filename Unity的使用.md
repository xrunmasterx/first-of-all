# Unity的使用

1. 右键Hierarchy可创建3D模型
2. 3D模型的使用及编辑

脚本

| 类型        | 作用         |
| ----------- | ------------ |
| Transform   | 调整模型位置 |
| Mesh Filter | 渲染表面涂层 |
| Rigidbody   | 使模型有重力 |



3. UI的创建及使用

   UI种类

| 类型   | 作用                       |
| ------ | -------------------------- |
| Text   | 编辑显示文字               |
| Image  | 显示图片                   |
| Button | 可以控制某一脚本的运行     |
| Toggle | 可控制是否勾选使用某一功能 |
